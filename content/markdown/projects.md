### Random Tables

A set of tools to modularise thematic randomness, by generating a collection of thesarus's that can be chained through a syntax to add depth to random generating tables.

I also created a desktop app with a large selection of random tables, leveraging Tauri to create binaries as small as 3Mb by leveraging Tauri.
